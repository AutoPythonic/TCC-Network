GitHub Pages Instructions

Since this site is hosted on GitHub Pages, you need to manually update the status.json file in your repository:

  1. Make your changes using the toggles above
  2. Click "Save Status Changes" to generate the updated JSON
  3. Copy the generated JSON
  4. Update the data/status.json file in your GitHub repository with this content
  5. Commit and push the changes to your repository
  6. Wait a few minutes for GitHub Pages to update (typically 1-2 minutes)

Note: All users will see the updated status after GitHub Pages refreshes your site.
